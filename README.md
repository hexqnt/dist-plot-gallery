# Графики распределений GAMLSS

Скрипт генерирует иллюстративные графики всех одномерных распределений и естественных параметризаций из `gamlss-family` в стиле справочных страниц: несколько кривых с разными значениями параметров, легенда и сетка. Для непрерывных распределений строится плотность, а для дискретных — функция вероятностей. Смешанные семейства (Tweedie, BEINF и ZAGA) показывают плотность и точечные вероятности на границах поддержки.

Каждое распределение сохраняется в отдельной папке в PNG и SVG:

```text
output/
  normal/
    normal.png
    normal.svg
  poisson/
    poisson.png
    poisson.svg
```

## Запуск

```bash
uv run python main.py --output-dir output --jobs 4 --dpi 300
```

- `--output-dir` — каталог для сгенерированных графиков;
- `--jobs` — число параллельных процессов (по умолчанию используются все доступные ядра);
- `--dpi` — разрешение только растровых PNG-файлов (по умолчанию `180`). SVG остаётся векторным.
- `--silence` — не показывать прогресс-бар.

Во время генерации отображается прогресс-бар с числом готовых графиков.
